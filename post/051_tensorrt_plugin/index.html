Recently, I had the chance to revisit the topic of deep learning model inference. It made me realize that the tools and frameworks have evolved a lot.

At a quick glance, optimizing LLM inference is pretty different from optimizing a typical CNN model that I used to work on quite a long time ago.

My north star goal is to document how to optimize LLM model inference for modern GPUs and support data precision like FP8 and INT8. This will probably span a month. So with this, I will break down the plan into a few steps or posts:

  1. Understand the TensorRT plugin
  2. Understand the computation & operations in LLM
  3. Tools and landscape of optimizing LLM model inference
  4. Optimize LLM model inference
    • Pick a popular LLM model
    • Optimize the inference; implement necessary custom op/CUDA kernel as necessary

0) TensorRT Plugin

To optimize model inference at a low level, we can identify operations in the graph that could be implemented as custom TensorRT plugins using CUDA kernels. This often requires replacing the operation in the ONNX graph with the custom plugin, which involves rewriting the ONNX graph.

In this section, I will dissect a simple operation called RoiAlign and its TensorRT plugin implementation. I will document how we can use this custom op in ONNX file and run inference using TensorRT.

Specifically, I will:

  1. Implement a custom TensorRT plugin
  2. Create an ONNX file with the custom operation
  3. Run inference

1) RoiAlign TensorRT plugin

enqueue

IPluginV3OneRuntime::enqueue is the function that gets called to execute the operation. The inputDesc and outputDesc are the tensor descriptor, the inputs and outputs are input and output tensors. The stream is the stream that the kernel will execute.

Note how we need to handle the type in switch case and cast the tensors accordingly.

int32_t ROIAlignCustom::enqueue(PluginTensorDesc const* inputDesc,
                                PluginTensorDesc const* outputDesc,
                                void const* const* inputs, void* const* outputs,
                                void* workspace, cudaStream_t stream) noexcept {
    // ...

    // No-op pass-through for empty ROIs
    if (mROICount == 0) {
        return 0;
    }

    auto type = inputDesc[0].type;

    // ...

    switch (type) {
        case nvinfer1::DataType::kFLOAT: {
            auto bottomData = static_cast<float const*>(inputs[0]);
            auto bottomRois = static_cast<float const*>(inputs[1]);
            auto batchIndicesPtr = static_cast<int32_t const*>(inputs[2]);
            auto topData = static_cast<float*>(outputs[0]);

            return RoiAlignImpl<float>(
                stream, mMaxThreadsPerBlock, bottomData, mSpatialScale,
                mROICount, mFeatureLength, mHeight, mWidth, mOutputHeight,
                mOutputWidth, mSamplingRatio, bottomRois, topData, mMode,
                batchIndicesPtr, mAligned);
        } break;
        case nvinfer1::DataType::kHALF: {
            auto bottomData = static_cast<__half const*>(inputs[0]);
            auto bottomRois = static_cast<__half const*>(inputs[1]);
            auto batchIndicesPtr = static_cast<int32_t const*>(inputs[2]);
            auto topData = static_cast<__half*>(outputs[0]);

            return RoiAlignImpl<__half>(
                stream, mMaxThreadsPerBlock, bottomData, mSpatialScale,
                mROICount, mFeatureLength, mHeight, mWidth, mOutputHeight,
                mOutputWidth, mSamplingRatio, bottomRois, topData, mMode,
                batchIndicesPtr, mAligned);
        } break;
        default:
            return -1;
    }

    return 0;
}

RoIAlign CUDA Kernel

The is the low level interesting part: RoiAlignImpl from the enqueue function above launches the kernel RoIAlignForward. I will get back to this part later if time allows.

template <typename T>
__global__ void RoIAlignForward(
    int32_t const nthreads, T const* bottomData, T const spatialScale,
    int32_t const channels, int32_t const height, int32_t const width,
    int32_t const pooledHeight, int32_t const pooledWidth,
    int32_t const samplingRatio, T const* bottomRois, T* topData,
    int32_t const isModeAvg, int32_t const* batchIndicesPtr,
    int32_t const aligned) {
    for (size_t index = blockIdx.x * blockDim.x + threadIdx.x; index < nthreads;
         index += blockDim.x * gridDim.x) {
        // (n, c, ph, pw) is an element in the pooled output
        int32_t pw = index % pooledWidth;
        int32_t ph = (index / pooledWidth) % pooledHeight;
        int32_t c = (index / pooledWidth / pooledHeight) % channels;
        int32_t n = index / pooledWidth / pooledHeight / channels;

        T const* offsetBottomRois = bottomRois + n * 4;
        auto const roiBatchInd = batchIndicesPtr[n];

        bool continuousCoordinate = aligned;
        // Do not using rounding; this implementation detail is critical
        T roiOffset = static_cast<T>(continuousCoordinate ? 0.5 : 0);
        T roiStartW = offsetBottomRois[0] * spatialScale - roiOffset;
        T roiStartH = offsetBottomRois[1] * spatialScale - roiOffset;
        T roiEndW = offsetBottomRois[2] * spatialScale - roiOffset;
        T roiEndH = offsetBottomRois[3] * spatialScale - roiOffset;

        T roiWidth = roiEndW - roiStartW;
        T roiHeight = roiEndH - roiStartH;
        if (!continuousCoordinate) {  // backward compatiblity
            // Force malformed ROIs to be 1x1
            roiWidth = floatMax(roiWidth, static_cast<T>(1.));
            roiHeight = floatMax(roiHeight, static_cast<T>(1.));
        }
        T binSizeH = static_cast<T>(roiHeight) / static_cast<T>(pooledHeight);
        T binSizeW = static_cast<T>(roiWidth) / static_cast<T>(pooledWidth);

        T const* offsetBottomData =
            bottomData +
            static_cast<int32_t>((roiBatchInd * channels + c) * height * width);

        // We use roiBinGrid to sample the grid and mimic integral
        int32_t roiBinGridH;
        if (samplingRatio > 0) {
            roiBinGridH = samplingRatio;
        } else {
            roiBinGridH = ceilf(roiHeight / static_cast<T>(pooledHeight));
        }

        int32_t roiBinGridW;
        if (samplingRatio > 0) {
            roiBinGridW = samplingRatio;
        } else {
            roiBinGridW = ceilf(roiWidth / static_cast<T>(pooledWidth));
        }
        // We do average (integral) pooling inside a bin
        T const count = roiBinGridH * roiBinGridW;  // e.g. = 4

        T const yOff = roiStartH + static_cast<T>(ph) * binSizeH;
        T const yFac = binSizeH / static_cast<T>(roiBinGridH);

        T const xOff = roiStartW + static_cast<T>(pw) * binSizeW;
        T const xFac = binSizeW / static_cast<T>(roiBinGridW);

        T outputVal = 0.;
        bool maxFlag = false;
        for (int32_t iy = 0; iy < roiBinGridH; iy++)  // e.g., iy = 0, 1
        {
            T const y =
                yOff + static_cast<T>(iy + .5F) * yFac;  // e.g., 0.5, 1.5
            for (int32_t ix = 0; ix < roiBinGridW; ix++) {
                T const x = xOff + static_cast<T>(ix + .5F) * xFac;

                T val = bilinearInterpolate(offsetBottomData, height, width, y,
                                            x, isModeAvg, index);

                if (isModeAvg) {
                    outputVal += val;
                } else {
                    if (!maxFlag) {
                        outputVal = val;
                        maxFlag = true;
                    } else {
                        outputVal = floatMax(outputVal, val);
                    }
                }
            }
        }
        if (isModeAvg) {
            outputVal = outputVal / count;
        }

        topData[index] = outputVal;
    }
}

template <typename T>
cudaError_t RoiAlignImpl(cudaStream_t stream, int32_t const maxThreadsPerBlock,
                         T const* bottomData, T const spatialScale,
                         int32_t const numRois, int32_t const channels,
                         int32_t const height, int32_t const width,
                         int32_t const pooledHeight, int32_t const pooledWidth,
                         int32_t const samplingRatio, T const* bottomRois,
                         T* topData, int32_t const isModeAvg,
                         int32_t const* batchIndicesPtr,
                         int32_t const aligned) {

    int32_t const outputSize = numRois * channels * pooledHeight * pooledWidth;

    int32_t blocksPerGrid = static_cast<int32_t>(
        ceil(static_cast<float>(outputSize) / maxThreadsPerBlock));

    RoIAlignForward<T><<<blocksPerGrid, maxThreadsPerBlock, 0, stream>>>(
        outputSize,       // nthreads
        bottomData,       // bottomData
        spatialScale,     // spatialScale
        channels,         // channels
        height,           // height
        width,            // width
        pooledHeight,     // pooledHeight
        pooledWidth,      // pooledWidth
        samplingRatio,    // samplingRatio
        bottomRois,       // bottomRois
        topData,          // topData
        isModeAvg,        // isModeAvg
        batchIndicesPtr,  // batchIndicesPtr
        aligned);

    return cudaGetLastError();
}

Generating Plugin Shared Library

TensorRT allows custom plugins to be registered either statically or dynamically. To create a plugin shared library, it must define the following public symbols:

extern "C" void setLoggerFinder(ILoggerFinder* finder);
extern "C" IPluginCreator* const* getCreators(int32_t& nbCreators) const;

This is very common when writing the plugins; these entry points are required to interface with the framework, enabling it to manage the plugin correctly.

2) Create an ONNX File with the Custom Operation

Replacing an operation in an ONNX graph with a custom one often requires rewriting the graph, typically with the help of the onnx_graphsurgeon Python library. However, in this section, I will show how to create a simple ONNX graph that directly uses the custom operation.

import os
import numpy as np
import onnx
import onnx_graphsurgeon as gs


input_tensor = gs.Variable(name='X', dtype=np.float32, shape=[1, 3, 8, 8])
rois_tensor = gs.Variable(name='rois', dtype=np.float32, shape=[2, 4])
batch_indices_tensor = gs.Variable(name='batch_indices', dtype=np.int32, shape=[2])

output_tensor = gs.Variable(name='Y', dtype=np.float32, shape=[2, 3, 2, 2])

node_roialign = gs.Node(
    op="ROIAlignCustom_TRT",
    inputs=[input_tensor, rois_tensor, batch_indices_tensor],
    outputs=[output_tensor],
    attrs={
        "mode": 1,  # 'avg' (1) or 'max' (0)
        "output_height": 2,
        "output_width": 2,
        "sampling_ratio": 2,
        "spatial_scale": 1.0
    }
)

graph = gs.Graph(nodes=[node_roialign], inputs=[input_tensor, rois_tensor, batch_indices_tensor], outputs=[output_tensor])
model = gs.export_onnx(graph)

onnx.save(model, os.path.join(output_dir, f"{output_name}.onnx"))

2) Run Inference

To build the TensorRT engine from the ONNX model containing custom operations, we will need to load the shared library for use with the builder.

import tensorrt as trt

TRT_LOGGER = trt.Logger(trt.Logger.WARNING)
PLUGIN_LIBRARY_PATH = "path_to_plugin_shared_library.so"

trt.get_plugin_registry().load_library(plugin_path=PLUGIN_LIBRARY_PATH)

builder = trt.Builder(TRT_LOGGER)
network = builder.create_network(0)
parser = trt.OnnxParser(network, TRT_LOGGER)

# Importing a Model Using the ONNX Parser
if not parser.parse_from_file(onnx_path):
    raise RuntimeError("Failed to parse ONNX file")
# ...

# Building an Engine
config = builder.create_builder_config()
serialized_engine = builder.build_serialized_network(network, config)
with open(os.path.join(output_dir, "custom_op.engine"), "wb") as f:
    f.write(serialized_engine)

# Deserializing a Plan
runtime = trt.Runtime(TRT_LOGGER)
engine = runtime.deserialize_cuda_engine(serialized_engine)

# Performing Inference
# ...
